My Python RPG ⚔️
Это моя первая серьезная игра на Python с графическим интерфейсом. Проект прошел путь от текстового квеста в консоли до полноценного оконного приложения с разделением логики и сохранением прогресса.
🎨 Основные механики:

    Современный GUI: Интерфейс построен на библиотеке CustomTkinter с поддержкой темной темы и закругленными кнопками.
    Модульная система: Логика разделена на независимые файлы (main, shop, hunt, casino) для удобного управления.
    Интерактивная охота: Пошаговые сражения с монстрами (Слизень, Волк, Огр) прямо в графических окнах.
    Система торговли: Покупайте снаряжение, которое сразу увеличивает ваш урон. Оружие покупается один раз, зелья — сколько угодно.
    Казино "Удачный Слизень": Испытайте удачу через графические диалоги и удвойте золото.
    Живой инвентарь: Кнопочная система использования предметов. Зелья мгновенно восстанавливают HP и "воскрешают" героя.
    Система сохранений: Весь прогресс (имя, HP, золото, вещи) записывается в save.json.
    Визуализация: Обработка иконок предметов через библиотеку Pillow.

🛠 Технологический стек:

    Язык: Python 3.10+
    Интерфейс: CustomTkinter
    Графика: PIL (Pillow)
    Данные: JSON
    ОС: Разработка и тестирование велись на Arch Linux 🐧

    bash

    Используйте код с осторожностью.

    Используйте код с осторожностью.

📈 История разработки:

    Консольный этап: Реализация пошагового боя с циклом while и текстового меню.
    Модульность: Полный рефакторинг кода и вынос логики в отдельные модули.
    Переход на GUI: Полная замена input() и print() на графические окна, кнопки и диалоги.
    Стабильность: Настройка корректных путей через BASE_DIR и устранение конфликтов с системным кэшем Python.
    UI/UX: Добавление системы блокировки кнопок при смерти и кнопки быстрого рестарта игры.
