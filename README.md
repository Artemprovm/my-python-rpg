My Python RPG ⚔️
Это моя первая серьезная игра на Python с графическим интерфейсом. Проект прошел путь от простого текстового квеста до полноценного оконного приложения с архитектурой «клиент-база данных».
🎨 Основные механики:

    Современный GUI: Интерфейс на CustomTkinter с адаптивной темной темой.
    Модульная логика: Код разделен на независимые модули (shop, hunt, casino), что упрощает расширение игры.
    Система многослотовых сохранений: Игрок может вести несколько параллельных прохождений. Выбор слота реализован через отдельное графическое меню при запуске.
    Интерактивная охота: Пошаговые сражения с монстрами (Слизень, Волк, Огр) с мгновенным обновлением полоски здоровья.
    Торговля и развитие: Покупка снаряжения, перманентно повышающего урон, и расходников.
    Казино «Удачный Слизень»: Мини-игра на удачу с графическими диалогами.
    Живой инвентарь: Использование предметов кнопками с мгновенным визуальным откликом.

🛠 Технологический стек:

    Язык: Python 3.10+
    Интерфейс: CustomTkinter
    База данных: SQLite3 (хранение статов, инвентаря и прогресса в разных слотах)
    Графика: PIL (Pillow) для обработки иконок
    ОС: Разработка и тестирование — Arch Linux 🐧

📈 История разработки:

    Консольный этап: Базовая логика боя и циклы.
    Переход на SQL: Отказ от save.json в пользу надежной базы данных SQLite.
    GUI Революция: Замена всех input() на красивые диалоговые окна и кнопки.
    Система слотов: Реализация стартового меню выбора персонажа и логики сохранения в разные ячейки базы.
