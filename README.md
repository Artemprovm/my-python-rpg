# My Python RPG ⚔️

Это моя первая серьезная игра на Python, созданная за 2 дня обучения. Проект прошел путь от одного файла до модульной архитектуры с разделением логики.

### 🎮 Основные механики:
* **Модульная система:** Игра разделена на файлы (`main`, `shop`, `hunt`, `casino`) для удобного управления кодом.
* **Пошаговая охота:** Сражайтесь с монстрами (Слизень, Волк, Огр) в бесконечном цикле боя до победы.
* **Система торговли:** Покупайте снаряжение, которое сразу увеличивает ваш урон. Зелья можно покупать стаками!
* **Казино "Удачный Слизень":** Испытайте удачу и удвойте золото (или проиграйте всё).
* **Инвентарь:** Интерактивное использование предметов (лечение зельями через быстрые команды "да"/"нет").
* **Графика:** Визуализация предметов через библиотеку Pillow при покупке.

### 🛠 Технологический стек:
- **Язык:** Python 3
- **Библиотеки:** `PIL` (Pillow), `os`, `random`
- **Архитектура:** Функциональное программирование и модульность
- **ОС:** Разработка и тестирование велись на **Arch Linux**

### 📦 Как запустить:
1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com
   ```
2. Установите зависимости:
   ```bash
   pip install Pillow
   ```
3. Запустите главный файл:
   ```bash
   python main.py
   ```

### 📈 История разработки:
- Полный рефакторинг кода и вынос логики в отдельные модули.
- Реализация системы пошагового боя с циклом `while`.
- Настройка корректных путей через `BASE_DIR` для работы на любых ОС.
- Ограничение на покупку уникальных предметов (оружие — 1 шт, зелья — бесконечно).
- Добавлен Класс всех переменных
