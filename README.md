# My Python RPG ⚔️

Это моя первая текстовая RPG на Python.

### Что уже есть в игре:
* **Торговля:** можно покупать снаряжение.
* **Инвентарь:** отслеживание вещей и золота.
* **Информация:** подробные характеристики предметов.

### Как запустить:
1. Установите Python.
2. Запустите файл командой: `python RPG.py`
