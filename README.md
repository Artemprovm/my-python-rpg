# My Python RPG ⚔️

[Русский](#русский) | [English](#english) | [日本語](#日本語)

---

## Русский

Это моя первая серьезная игра на Python с графическим интерфейсом. Проект прошел путь от простого текстового квеста до полноценного оконного приложения.

### 🎨 Основные механики:
* **Современный GUI:** Интерфейс на CustomTkinter с поддержкой локализации.
* **Локализация:** Полная поддержка RU/EN/JP языков, переключаемых «на лету».
* **Система сохранений:** SQLite3 для хранения прогресса в нескольких слотах.
* **Мини-игры:** Охота на монстров, Магазин и Казино «Удачный Слизень».

### 🛠 Стек:
* **Python 3.10+**, **CustomTkinter**, **SQLite3**, **Pillow**.

### 📦 Запуск:
```bash
git clone https://github.com
pip install customtkinter pillow
python main.py
```

## English

A full-featured RPG game built with Python, featuring a modern graphical interface and multi-language support.

### 🎨 Key Features:
* **Modern GUI:** Built with CustomTkinter for a sleek look.
* **Localization:** Full RU/EN/JP support with real-time switching.
* **Save System:** Multi-slot progress tracking using SQLite3.
* **Activities:** Monster hunting, gear shopping, and "Lucky Slime" Casino.
